# Word Sense Disambiguation using Lesk Algorithm

## Overview

This project demonstrates **Word Sense Disambiguation (WSD)** using the **Lesk Algorithm** implemented in the Natural Language Toolkit (NLTK). The program identifies the correct meaning (sense) of an ambiguous word based on its surrounding context using the WordNet lexical database.

## Features

* Tokenizes input text using NLTK.
* Performs Word Sense Disambiguation with the Lesk algorithm.
* Retrieves the most appropriate WordNet synset.
* Displays the definition of the selected word sense.
* Simple and beginner-friendly Python implementation.

## How It Works

* The input text is tokenized into individual words.
* The Lesk algorithm analyzes the context surrounding the target word.
* WordNet is used to compare possible meanings (synsets) of the word.
* The best matching sense is selected and its definition is displayed.

## Requirements

- Python 3.x
- NLTK
